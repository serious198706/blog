Android三方库Retrofit

Retrofit 实现原理解析

2023-03-23

Retrofit 是现在比较常用的网络请求库了，它的可扩展性强，底层网络请求集成了 Okhttp，异步处理可集成 RxJava，内容解析可集成 Gson，Jackson 等。而且全面支持 Restful 请求，并且通过注解的方式，支持链式调用，使用简洁方便。精妙的源码设计模式，内部层次分工明确，解耦性强。 它的使用核心是实现一个 Java 接口，用注解的方式来指定各种网络请求中的字段、方法等。
